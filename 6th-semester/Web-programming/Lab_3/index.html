Задание 1

С помощью функций ввод/вывод данных и диалоговых окон необходимо выполнить:
Запросите ввод времени в часах и минутах. Определите время, которое будет через минуту. Предусмотреть три случая:
  1. увеличение только минут;
  2. увеличение числа часов, обнуление числа минут;
  3. переход через полночь (00:00).

Задание 2

Вывести на html-страницу:
По аналогии и в соответствии с примером из раздела «Пример №3. Использование метода document.write()» из закрашенных ячеек таблицы построить расстановку шашек на доску 8 на 8. Чёрные справа.

Задание 3

Выполнить задание:

Выполнить задание:
